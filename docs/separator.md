# Separator

| property | description                                   | type                                     | default  | required  |
| -------- | --------------------------------------------- | ---------------------------------------- | -------- | --------- |
| size     | Sets the size of the space (width and height) | **"xs" \| "sm" \| "md" \| "lg" \| "xl"** | **"md"** | **false** |

### Usage

```js
<Separator size="xl" />
```
